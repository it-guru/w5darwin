Marvin Technologie

Marvin setzt auf J5Base - einer JavaScript Erweiterung für den Zugriff auf W5Base Applikationen. Zusätzlich kommen die folgenden JavaScript Erweiterungen zum Einsatz:

  • jQuery
  • jQuery-Mobile
  • i18next

Das Haupt Interface erwartet keine Authentifikation und kann somit keinerlei Unterscheidung über die Sichtbarkeit von Menüpunkten machen (d.h. jeder sieht auf dieser Ebene alles). Erst in den einzelnen Untermodulen werden wird bei Bedarf eine Authentifikation angefordert. Von da an können dann bestimmte Funktionen für einzelne Usergruppen eingeschränkt sein.

Alle Marvin Anwendungen sind von ihrer Kern-Struktur gleich aufgebaut.
