Programmierung: Perl5
Was erhalten Sie auf die Standard-Ausgabe (oder Standard-Error) dargestellt, wenn Sie den folgenden Programmcode ausführen: 
#!/usr/bin/env perl
my $v; 
if ($v->{myKey} ne ""){
   print("keyFound\n");
}
print $v."\n";

Systemadministration: Grundwissen
Starten Sie eine Texteditor (vorzugsweise VI):
  1. Geben Sie den Text "Dies ist ein Test Text ein"
  2. Kopieren Sie die eingegebene Zeile 15 mal
  3. Entfernen Sie aus den Zeilen 3-11 das Wort "Test" möglichst effizient
  4. Speichern Sie den Text unter dem Namen "test.txt" im DOS Format
