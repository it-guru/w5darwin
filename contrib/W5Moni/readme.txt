CheckIT kann ein W5Base System monitoren. Es werden sowohl
Performance als auch Verfügbarkeitsdaten gemessen.

In der *.conf Datei können die Parameter des Checks festgelegt
werden (sample.conf sollte eigentlich sprechend sein).

Mit CheckIT --help bekommt man die Hilfe angezeigt.

Es muß auf dem CheckServer min. die W5API Version 1.1 installiert
sein.
